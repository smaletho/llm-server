# Tiny GPT-2 Model

This directory contains the Tiny GPT-2 model files.

## How to Download

1. Visit the Tiny GPT-2 model on Hugging Face:
   https://huggingface.co/microsoft/DialoGPT-small

2. Download the following files:
   - config.json
   - pytorch_model.bin
   - tokenizer_config.json
   - vocab.json
   - merges.txt

3. Place all downloaded files in this directory.

Note: Model files are not included in the repository due to their large size.
